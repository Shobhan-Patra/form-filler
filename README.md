# Form Filler Extension

A simple Chrome extension that helps you autofill repetitive job and web forms with a single click.

## Features:
- Detects common form fields such as name, email, phone, and skills.
- Simple design for easy navigation and setup.
- Autofill forms directly from the toolbar icon.
- Works on most standard HTML forms.

## How to setup locally:
1. Clone this repo into your machine and open it
```
git clone git@github.com:Shobhan-Patra/form-filler.git
```

2. Load this extention in Chrome  
    a. Open Chrome  
    b. Go to `chrome://extensions`  
    c. Turn on Developer Tools (Top right toggle button)  
    d. Click `Load unpacked` button  
    e. Select the downloaded form-filler folder  

## How to use
1. Open any webpage containing a form
2. Click the Form Filler extension icon in the toolbar
3. Press the fillForm button
4. Supported fields will be filled automatically

## Supported Fields

- Text inputs (e.g. name, skills)
- Email inputs
- Phone number input
